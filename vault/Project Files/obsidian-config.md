# Obsidian Config (.obsidian/)

## Overview

ספריית ההגדרות של Obsidian — נוצרת אוטומטית כאשר פותחים את ספריית הפרויקט כ-vault ב-Obsidian. **אין לערוך ידנית** — Obsidian מנהל קבצים אלה.

| קובץ | תוכן |
|---|---|
| `app.json` | הגדרות אפליקציית Obsidian |
| `appearance.json` | הגדרות ערכת נושא ומראה |
| `core-plugins.json` | plugins מובנים מופעלים/כבויים |
| `workspace.json` | מצב חלונות ולשוניות פתוחות |

## מיקום

`/.obsidian/` — שורש הפרויקט (שורש ה-vault)

## Open Questions

- לשקול הוספת `.obsidian/` ל-`.gitignore` — הגדרות הן לוקליות ועלולות לגרום לקונפליקטים בין מחשבים
