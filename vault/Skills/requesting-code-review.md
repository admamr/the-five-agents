# Requesting Code Review

## Overview

Skill לבקשת code review לאחר השלמת משימה. מנחה כיצד להכין את הקוד לביקורת, מה לכלול בבקשה, ואיך לשלוח ל-reviewer. מכיל קובץ `code-reviewer.md` שמגדיר את תפקיד ה-reviewer sub-agent.

## קבצים משויכים

| קובץ | תפקיד |
|---|---|
| `SKILL.md` | פרוטוקול בקשת review |
| `code-reviewer.md` | פרומפט ל-reviewer sub-agent |

## מתי להשתמש

לאחר השלמת feature, מימוש משמעותי, או לפני merge — לאימות שהעבודה עומדת בדרישות.
