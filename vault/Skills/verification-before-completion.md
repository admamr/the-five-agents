# Verification Before Completion

## Overview

Skill שמחייב הרצת פקודות אימות ובדיקת תוצאותיהן לפני הצהרה שהעבודה הושלמה. "Evidence before assertions" — ראיות לפני טענות. מונע commit או PR על בסיס הנחות בלבד.
