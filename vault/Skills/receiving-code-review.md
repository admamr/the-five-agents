# Receiving Code Review

## Overview

Skill לקבלת משוב code review עם ביקורתיות טכנית. מדגיש אי-הסכמה רפלקטיבית — לא לקבל כל הצעה עיוורת, אלא לאמת תחילה שהמשוב נכון טכנית לפני מימוש.

## קבצים משויכים

| קובץ | תפקיד |
|---|---|
| `SKILL.md` | פרוטוקול קבלת ועיבוד משוב review |

## מתי להשתמש

כאשר מקבלים משוב code review, במיוחד אם ההצעות נראות לא ברורות או שנויות במחלוקת.
