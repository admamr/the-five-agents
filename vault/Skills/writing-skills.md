# Writing Skills

## Overview

Meta-skill ליצירה, עריכה ואימות של skills חדשים לפני deploy. מגדיר מבנה SKILL.md (frontmatter עם name ו-description), כיצד לנסח trigger descriptions, ואיך לבדוק שה-skill מופעל נכון. כולל best practices של Anthropic, עקרונות שכנוע, ודוגמאות.

## קבצים משויכים

| קובץ | תפקיד |
|---|---|
| `SKILL.md` | פרוטוקול יצירת skills |
| `anthropic-best-practices.md` | best practices של Anthropic |
| `persuasion-principles.md` | עקרונות שכנוע לשימוש ב-skills |
| `graphviz-conventions.dot` | קונבנציות לדיאגרמות Graphviz |
| `render-graphs.js` | סקריפט רינדור גרפים |
| `testing-skills-with-subagents.md` | בדיקת skills עם sub-agents |
| `examples/CLAUDE_MD_TESTING.md` | דוגמת בדיקת CLAUDE.md |

## מתי להשתמש

כאשר יוצרים skill חדש, עורכים skill קיים, או בודקים שה-skill עובד.
