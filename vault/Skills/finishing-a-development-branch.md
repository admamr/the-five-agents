# Finishing a Development Branch

## Overview

Skill לסיום ענף פיתוח לאחר שהמימוש הושלם וכל הטסטים עוברים. מציג אפשרויות מובנות למיזוג: PR, merge ישיר, squash, או ניקוי — ומנחה בבחירה המתאימה לפי הקונטקסט.

כאשר המימוש הושלם, הטסטים עוברים, וצריך להחליט איך לשלב את העבודה ב-main.
